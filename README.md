# Kacha Bazaar

## Description

This project intends to serve a node application backend for Kacha Bazaar Application.

## Installation

To install this module to an external project, follow the procedure:

1. **npm install** or **yarn install**

2. Create a `serviceAccountKey.json` file in the project directory to store your Firebase config.

## Usage

1. Run **npm start** and observe the console.
